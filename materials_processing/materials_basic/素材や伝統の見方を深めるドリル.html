素材や伝統の見方を深めるドリル

木材を扱うときの視点

ちょっと気になるひとこと
「木って年輪があったり、節があったりで手間かかるし、人工素材の方がわかりやすくて安心だよね」

STEP1｜あなたはどう思う？

1. なるほど、合理的な考え方かも
2. 木には木の価値があると思う
3. なんとなく受け入れてしまいそうだった
4. 正直、判断が難しい
→ 効率を重視する考え方には「自然素材＝不便」という前提があるかもしれません。
→ 木材の背景や育った環境、人の手に思いを馳せる視点が含まれています。
→ 「なんとなく」はよくある反応。そこから考えることが大事です。
→ 判断がつかないのは問いを持てた証。じっくり考えてみましょう。

STEP2｜問い返してみる ▼

  • ・人工素材って、どんな場面で求められてるんだろう？
  • ・燃やすって…その木はどこから来たんだろう？

どんな構造がある？

  • ・“扱いやすさ＝価値”という偏った見方になっていないか？
  • ・複雑さや地域性のある素材を排除する構造では？

STEP4｜行動につなげる問い

→ 木を選ぶとき、何を優先したいと思う？

解説
自然素材は不均一で手間がかかるというのは一面の事実。しかし、それが「価値がない」「非合理」とするのは誤謬です。地元の木材には、風土・歴史・人の手が重なっています。人工素材との対比ではなく、目的や役割に応じて素材を選ぶ視点が必要です。

地元の間伐材と補助金の話

ちょっと気になるひとこと
「地元材って丸太から板に製材したままだと、同じおおきさの板でもはえていたところで全然違うし、助金付きの輸入合板使えば楽でしょ」

STEP1｜あなたはどう思う？

1. 効率重視なら補助金付きがいいと思う
2. 地元材にも意味があるのでは？
3. 言われてみると…判断に迷う
4. 補助金って誰のための制度なんだろう？
→ 効率の陰にある価値や背景を見逃してしまうかもしれません。
→ 地元の森林や人との関係を考える第一歩です。
→ 迷いは大事な問いの始まり。何を大事にするかが問われています。
→ 「制度の目的」を見直すことは、社会構造への問いになります。

STEP2｜問い返してみる ▼

  • ・補助金って、素材の“意味”まで決めてしまうもの？
  • ・将来の森づくりは、誰がどう考えている？

STEP3｜どんな構造がある？

  • ・“補助金が出る＝使う価値が高い”というすり替え
  • ・地域との関係性が切れてしまう制度的な問題

STEP4｜行動につなげる問い

→ 木材選びで、山や地域との未来をつなげるには？

解説
補助金や制度によって選ばれる素材は、経済合理性で選ばれた結果です。 しかし、地元材には森林管理や地域の産業との関係があります。補助金の有無だけでなく、「誰のための素材選びか」という視点が重要です。

伝統をどうとらえる？

ちょっと気になるひとこと
「伝統って何？　なんか関係ない気がするし。昔からやってることなんだから、理由なんて考えなくていいんじゃない？」

STEP1 あなたはどう思う？

1. たしかに、理由なんていらないかも
2. それって少し思考停止では…？
3. 守る理由や背景を知らないまま続けるのは不安
4. 伝統の“意味”って何だろう
→ 「昔からある」は魅力ですが、考えることを止めると意味が置き去りになるかも。
→ 素直な疑問は伝統に向き合う力になります。
→ 背景を理解することが、受け継ぐ責任を育てます。
→ 意味に目を向けることで、伝統は「生きた力」になります。

STEP2 問い返してみる ▼

  • ・なぜそれが受け継がれてきたのか、考えたことはある？
  • ・今の時代に合う形にアレンジする道はない？

STEP3 どんな構造がある？

  • ・“昔からある”＝“正しい”という誤謬
  • ・背景の意味や意図を問い直す視点の欠如

STEP4 行動につなげる問い

→ 自分なりに残したい伝統って、どんなもの？

解説
伝統には多くの知恵や工夫が詰まっていますが、それを「理由もわからず守る」だけでは、未来にはつながりません。 背筋が伸びるような感覚、それは“意味”を考える力が芽生えた証です。伝統を背負って未来へつなぐための姿勢が大切です。
