学びの約束

「技術」は人と人々の暮らしと自然をつなぐ仕組みを学ぶ科目です。

「科学」の根拠と「倫理」（りんり・地域や学校など世の中の一員として、何が正しくて何が間違っているかを考え、より良い人々の暮らしを作るために、みんなで守るべきルールや考え方）の価値判断を混同しないで考えます。

「正解が一つとは限らない問い」を自分と地域に当てはめます。

便利さとリスクを比べて「意思決定できる力」を育てます。

このドリルは、物語と問いを通して「問いを止めない力」を育てます。

学びの約束をいつでも見返して、自分の暮らしに必要な技術を考え続けるようにしましょう。
